# BMLD-Inf2-AppV0

# Chemie ohne Kopfschmerzen

*Chemie ohne Kopfschmerzen* ist eine interaktive Lern- und Rechen-App für chemische Grundlagen wie Molrechnung, Konzentration und mit einem Periodensystem.

Sie kombiniert kompakte Theorie mit praktischen Tools und ermöglicht das Speichern von Ergebnissen im Logbuch.
